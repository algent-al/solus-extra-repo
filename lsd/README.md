### Aliases for LSD (ls deluxe). Listing directory contents.
After installing lsd it is recommended to add these aliases:

```
alias ls='lsd'
alias list='ls'
alias ll='ls -l'
alias la='ls -a'
alias lla='ls -la'
alias l1='ls -1'
```
